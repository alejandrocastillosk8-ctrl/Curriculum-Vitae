# 📄 Curriculum Vitae Profesional

Este proyecto consiste en el desarrollo de un **Curriculum Vitae y Portafolio Web** utilizando tecnologías modernas de maquetación. Es parte de las actividades del programa **ADSO** para demostrar habilidades en diseño responsivo y control de versiones.

## 🛠️ Tecnologías utilizadas
* **HTML5**: Estructura semántica.
* **CSS3**: Estilos personalizados.
* **Bootstrap 5**: Framework para diseño responsivo y componentes.
* **Git/GitHub**: Control de versiones y despliegue en GitHub Pages.

## 📁 Contenido
* **CV Principal**: Datos personales, educación y experiencia laboral.
* **Portafolio**: Sección externa con galería de proyectos realizados (Matita d' Arte, MySQL, etc.).
* **Sección de Contacto**: Integración con enlaces directos.
